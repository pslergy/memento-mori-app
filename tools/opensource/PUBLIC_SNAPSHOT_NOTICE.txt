Public source snapshot of Memento Mori
==========================================

This folder was assembled by the scripts/export_public_mirror script from the PRIVATE repository.

What has been removed
Sensitive modules (mesh, BLE transport, gossip, tunnels, native Android code, etc.) have been stripped according to the manifest in the private repo: tools/opensource/exclude-from-public.txt

All .md files (internal reports, README files in subfolders, etc.) have also been removed from the public copy.

Build
A full build (flutter build) is not expected to work here: Dart/Kotlin files are missing, and there may be leftover entries in AndroidManifest.xml (e.g., mesh services). Full builds only work in the private tree.

If you want to make it sound more technical or more user-friendly, let me know. I can also format it as a README.md for the public mirror folder if needed.
